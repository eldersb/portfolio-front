# Portifolio (Front-end)

Repositório contendo o front-end do meu portifólio.

## Tecnologias utilizadas
* HTML
* CSS (Bootstrap)
* JS
* Vue
* Vite

Clone o repositório para sua máquina e execute os seguintes comandos em sequência:

```
npm install
npm run dev
```
